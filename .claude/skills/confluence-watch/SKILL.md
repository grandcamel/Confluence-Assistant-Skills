# Confluence Watch Skill

Manage content watching and notifications.

## Available Scripts

### watch_page.py
Start watching a page.

### unwatch_page.py
Stop watching a page.

### watch_space.py
Watch an entire space.

### get_watchers.py
List content watchers.

### am_i_watching.py
Check if you're watching content.
