# Browse Confluence Skills

Display a categorized list of all available Confluence automation skills.

## Instructions

Read and present the skill catalog. Group skills by category and show:
- Skill name
- Brief description
- Example use case

## Output Format

Present as a formatted table or list showing:

### Content Management
- **confluence-page** - Create, read, update, delete pages and blog posts
- **confluence-space** - Space management and configuration
- **confluence-template** - Create pages from templates

### Search & Discovery
- **confluence-search** - CQL queries, saved filters, and export
- **confluence-label** - Add, remove, and search by labels
- **confluence-hierarchy** - Navigate page trees, ancestors, descendants

### Collaboration
- **confluence-comment** - Page and inline comments
- **confluence-attachment** - Upload, download, manage files
- **confluence-watch** - Watch/unwatch content for notifications

### Administration
- **confluence-permission** - Space and page permissions
- **confluence-property** - Custom metadata management
- **confluence-analytics** - Views, popularity, watchers

### Integration
- **confluence-jira** - Embed JIRA issues, cross-product links

### Meta
- **confluence-assistant** - Routes to the right skill automatically

---

For detailed info on any skill, use: `/skill-info <skill-name>`
